## Deskripsi Proyek
**Sistem Bilangan** adalah sebuah program sederhana berbasis **Python** yang digunakan untuk melakukan **konversi bilangan desimal** ke berbagai sistem bilangan lainnya, yaitu:
- **Biner (Basis 2)**
- **Oktal (Basis 8)**
- **Desimal (Basis 10)**
- **Heksadesimal (Basis 16)**

Program ini membantu pengguna memahami konsep dasar sistem bilangan yang digunakan dalam komputer serta cara konversinya.  
Melalui aplikasi ini, pengguna dapat belajar bagaimana komputer merepresentasikan data menggunakan berbagai sistem bilangan.
